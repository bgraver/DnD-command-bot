# DnD Command Bot
## Functionality:
- !info: Show info related to the bots functions
- !roll {1}d{2} {modifier}: Roll a {2} sided die {1} times, and then add the modifier
    - Example: !roll 2d20 +4 - Roll two 20 sided dice, and then add 4 to the total
